Services

Detailed documentation for all UAI microservices.

The UAI platform consists of six FastAPI microservices and a Python bridge adapter package.

Service Port Role
Registry 8000 Agent registration and discovery
Trust Server 8002 Token issuance and policy evaluation
Audit Ledger 8001 Immutable receipt storage
Demo Providers 8005 Five reference domain providers
Adapters — Bridge adapter and MCP bridge

Toy Provider (port 8003) and Mock Consent (port 8004) are development fixtures used in the steel thread and integration tests. They follow the same patterns as Demo Providers.

Registry

Agent registration, discovery, and proof-of-control validation.

Trust Server

Delegation token issuance, policy evaluation, and consent validation.

Audit Ledger

Append-only, tamper-proof receipt storage backed by PostgreSQL.

Demo Providers

Five reference domain providers covering agriculture, education, health, finance, and language.

Adapters

Bridge Adapter (DID resolution, discovery, negotiation, publishing) and MCP Bridge.
