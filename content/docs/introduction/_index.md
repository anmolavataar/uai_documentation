UAI (Unified AI Interface) is India's sovereign Digital Public Infrastructure for AI agent interoperability. It provides a trust and audit layer so that AI agents built by different organisations — government ministries, enterprises, hospitals, banks — can communicate securely, with citizen consent, and with a tamper-proof audit trail, all without ever sharing raw citizen data.
