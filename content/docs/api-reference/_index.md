All UAI services expose REST APIs built with FastAPI. Interactive OpenAPI documentation is available at `http://localhost:{port}/docs` for each running service.

| Service | Port | OpenAPI Docs |
|---------|------|-------------|
| Registry | 8000 | `http://localhost:8000/docs` |
| Audit Ledger | 8001 | `http://localhost:8001/docs` |
| Trust Server | 8002 | `http://localhost:8002/docs` |
| Demo Providers | 8005 | `http://localhost:8005/docs` |
