{{< blocks/section color="dark" type="row" >}}
{{% blocks/feature icon="fa-shield-alt" title="Trust Without Proximity" %}}
The UAI Trust Server issues short-lived delegation tokens so any two agents — across ministries, enterprises, or platforms — can call each other securely without sharing secrets.
{{% /blocks/feature %}}

{{% blocks/feature icon="fa-user-check" title="Citizen Consent First" %}}
Every sensitive data access requires a DPDP Act 2023-compliant consent artifact. UAI validates consent before issuing any token above Tier 1.
{{% /blocks/feature %}}

{{% blocks/feature icon="fa-file-alt" title="Tamper-Proof Audit Trail" %}}
Every interaction produces a cryptographically signed receipt stored in an append-only ledger. UAI stores hashes, never raw payloads — privacy by design.
{{% /blocks/feature %}}
